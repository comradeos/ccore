# C Learning Plan (K&R + Vim)

## Strategy

-   Read small parts
-   Write code after each section
-   Use vim only
-   Compile manually

------------------------------------------------------------------------

## Stage 1: Basics (Chapter 1)

-   printf, loops, structure
-   Practice: hello world, temperature table, char counter

## Stage 2: Types & Operators (Chapter 2)

-   int, char, float
-   if/else, switch
-   Practice: calculator, converters

## Stage 3: Flow Control (Chapter 3)

-   loops, break, continue
-   Practice: guessing game, sorting

## Stage 4: Functions (Chapter 4)

-   functions, recursion
-   Practice: factorial, modular programs

## Stage 5: Pointers & Arrays (Chapter 5)

-   pointers, memory
-   Practice: strlen, strcpy, reverse

## Stage 6: Structures (Chapter 6)

-   struct, arrays of structs
-   Practice: text analysis

## Stage 7: Input/Output (Chapter 7)

-   file handling
-   Practice: file copy, logs

## Stage 8: Build System

-   Makefile basics

## Stage 9: Debugging

-   lldb basics

## Stage 10: Projects

-   CLI calculator
-   grep clone
-   log analyzer

------------------------------------------------------------------------

## Daily Routine

1.  Read
2.  Code
3.  Break things
4.  Fix them
