инструменті: gdb - gnu debugger (отладчик), valgrind - инструмент для поиска утечек памяти, віход за грациці массива, чтения не инициализированной памяти (на макос не очень хорошо работает), lldb (macos) - то же что и gbd на линуксе

компиляция, флаги: 

gcc -Wall -Wextra -std=c17 -O0 -g main.c -o main

-Wall -Wextra - максимальніе предупреждения 
-std=c17 - стандарт язіка
-O0 - без оптимизации (что нзачит?)
-g - отладочная информация (примері?)

посмотреть размер
size <бинарник> 

Память:
STACK - функции, локальные переменные
HEAP - malloc / free
DATA - глобальные переменные
RODATA - строковые литералы (Литерал — это фиксированное значение, записанное прямо в коде: 10, 3.14, 'A', "hello".)
TEXT - машинный код (Машинный код — это набор числовых инструкций процессора)
